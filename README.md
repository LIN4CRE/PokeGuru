# PokeGuru

A fast, open Pokémon TCG card database.

PokeGuru is a zero-build web application that provides a modern interface for searching, browsing, and scanning Pokémon cards. It communicates directly with the public Pokémon TCG API to provide up-to-date card data, images, and market prices.

## Features

- 🔎 **Advanced Search** — Search cards by name or use field queries for precise results.
- 📷 **Card Scanner** — Identify cards instantly using your device camera or a photo.
- 🃏 **Set Browser** — Explore every official Pokémon TCG set, sorted by release date.
- 📄 **Rich Details** — View high-resolution artwork, attacks, abilities, and real-time market prices.
- 📱 **Mobile Ready** — Fully responsive dark theme optimized for all devices.

## Technology

Built with modern web standards:
- Vanilla JavaScript (ES6+)
- Dynamic CSS3 with View Transitions
- On-device OCR via Tesseract.js
- Data powered by [pokemontcg.io](https://pokemontcg.io)
